## 动画效果

### 安装animal.css

借助第三方的css工具实现动画效果`animate.css`

> 注意使用的时候 一定要加上 animate__animate 这个类名

```bash
 npm install animate.css --save
 ##在需要使用的地方加上类就行
 <h1 class="animate__animated animate__bounce">An animated element</h1>
```
