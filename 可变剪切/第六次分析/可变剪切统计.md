#### 统计AS与计算PSI值

SUPPA 

在`ioe`模式下生成的gtf文件用于在USC浏览器中加载；

```bash
conda activate suppa
##提取感兴趣的AS的坐标
suppa.py  generateEvents -i PacBio.gtf -o TM1 -e SE SS RI FL MX -f ioe 
```
