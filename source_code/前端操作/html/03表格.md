## 表格

基本语法

```html
 <table border="1">
<thead>
    <tr>
        <th>编号</th>
        <th>标题</th>
      </tr>
</thead>
<tbody>
  <tr>
    <td>dsadas</td>
    <td>dsadas</td>
  </tr>
</tbody>

 </table>
```

+ table 定义表格区域
+ thead
+ tbody
+ tfoot

### 单元格合并，不规则表格

+ rowspan ="合并行"
+ colspan=“合并列数目”

## 视频

+ 阿里云播放器 UI界面 拓展强
+ VIDEO.js

```html
<!-- vido 标签 --!>
<video src="" controls muted autoplay poster="C:\Users\zp\Desktop"></video>
```

+ controls 控制进度条
+ muted 静音
+ autoplay 自动播放
+ poster 封面
+ preload 视频加载方式，可以提前加载，也可以点击后在加载
+ loop 无限循环
