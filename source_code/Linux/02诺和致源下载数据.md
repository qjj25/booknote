### 1.下载脚本

```bash
wget http://api.data-deliver.novogene.com/welcome/shell?f=linux
#解压安装包
unzip shell?f=linux
#给可执行权限
cd linuxnd
chmod 755  linuxnd
```



### 2. 登录账户

```bash
./linuxnd -u 用户名 -p
```



### 3.下载数据

```bash
./software/novogene/linuxnd/linuxnd cp -d   oss://CP2018091202596/H101SC19090378/KY_kehu_JK/X101SC19090378-Z01/X101SC19090378-Z01-F001/ HPC22
```

+ -d 下载整个目录到本地
+ oss为云对象存储
+ 第三个为本地目录
